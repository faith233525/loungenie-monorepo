Production preview of the LounGenie Portal with Support & Partner dashboards.

== Description ==

LounGenie Portal Preview is a WordPress plugin that provides a complete production-ready preview of the LounGenie Portal. It includes:

- **Support Dashboard** - View all companies, units, and tickets
- **Partner Dashboard** - Company-scoped management interface
- **Support Login** - Outlook 365 authentication
- **Partner Login** - Username/password authentication
- **Service Request Form** - Complete request submission system
- **Role-Based Theming** - Cyan for Support, Teal for Partner
- **Responsive Design** - Works on all devices

This is a preview/demo plugin. For production use, install the full LounGenie Portal plugin.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through 'Plugins' menu in WordPress
3. Go to 'Portal Preview' menu in WordPress admin
4. Click 'Open Portal Preview' to view the complete portal
