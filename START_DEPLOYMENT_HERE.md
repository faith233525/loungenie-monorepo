### Step 1: Extract ZIP
```bash
# Extract to WordPress plugins directory
unzip loungenie-portal-1.8.1.zip -d /wp-content/plugins/
```

### Step 2: Activate Plugin
```bash
# Via WordPress Admin
Dashboard → Plugins → LounGenie Portal → Activate

# OR via WP-CLI
wp plugin activate loungenie-portal
```

### Step 3: Verify
```bash
# Navigate to WordPress
https://yoursite.com/wp-admin/plugins.php

# Should show: LounGenie Portal - Active
# Portal available at: /portal
```
