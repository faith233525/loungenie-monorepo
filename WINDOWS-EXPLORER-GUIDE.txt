                          🚀 DEPLOYMENT STEPS

════════════════════════════════════════════════════════════════════════════════

STEP 1: VERIFY
────────────────────────────────────────────────────────────────────────────
  Open PowerShell and run:
  php FINAL-20-TEST-SUITE.php

  Expected result: "20/20 TESTS PASSED ✅"

STEP 2: COPY
────────────────────────────────────────────────────────────────────────────
  Option A: Copy the folder
    cp -r . C:\path\to\wp-content\plugins\loungenie-portal

  Option B: Extract the zip
    Unzip loungenie-portal.zip to wp-content\plugins\

STEP 3: ACTIVATE
────────────────────────────────────────────────────────────────────────────
  In WordPress:
    Go to Plugins → Loungenie Portal → Activate
  
  OR in PowerShell:
    wp plugin activate loungenie-portal

STEP 4: CONFIGURE
────────────────────────────────────────────────────────────────────────────
  Set up these in WordPress admin:
    • Microsoft SSO credentials
    • HubSpot API keys
    • Email/Outlook settings
    • Company colors and branding

STEP 5: TEST & MONITOR
────────────────────────────────────────────────────────────────────────────
  Verify:
    • User login/logout works
    • Tickets can be created
    • Emails synchronize
    • API endpoints respond

  Monitor:
    • Check error logs daily
    • Monitor email sync crons
    • Watch database performance
    • For 7 days post-deployment
