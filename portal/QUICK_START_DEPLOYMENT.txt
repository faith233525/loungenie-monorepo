⚠️ TROUBLESHOOTING (IF SOMETHING DOESN'T WORK)
═══════════════════════════════════════════════════════════════════════

ISSUE: Plugin doesn't activate
─────────────────────────────
FIX:
  1. Check PHP version: PHP 7.4+ required
  2. Check WordPress version: 5.8+ required
  3. Check error logs: /wp-content/debug.log
  4. Verify file permissions: 755 for directories
  5. Try: Deactivate → Delete → Re-upload → Activate

ISSUE: Settings page doesn't appear
───────────────────────────────────
FIX:
  1. Hard refresh browser (Ctrl+Shift+R)
  2. Clear WordPress cache if using cache plugin
  3. Check admin debug log
  4. Verify you're logged in as admin

ISSUE: Map doesn't show
──────────────────────
FIX:
  1. Create at least one unit with address
  2. Clear browser cache
  3. Check JavaScript console (F12) for errors
  4. Verify units have coordinates assigned

ISSUE: Emails not sending
─────────────────────────
FIX:
  1. Check email settings: Portal Settings → Email
  2. Verify credentials are correct
  3. Check logs: /wp-content/uploads/lgp-logs/
  4. Try Graph API first, fallback to POP3
  5. Test with: wp-cli email test (if WP-CLI available)
