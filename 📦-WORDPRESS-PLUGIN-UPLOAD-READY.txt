✅ UPLOAD INSTRUCTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Prepare the Plugin
────────────────────────────

Option A - ZIP Upload (Easiest):
  1. Create folder: loungenie-portal
  2. Copy all items listed above into this folder
  3. ZIP the folder as: loungenie-portal.zip
  4. Size: ~2-3 MB

Option B - FTP Upload:
  1. Connect via FTP to your server
  2. Navigate to: /wp-content/plugins/
  3. Create folder: loungenie-portal
  4. Upload all items into this folder
  5. Maintain the folder structure shown above


STEP 2: Upload to WordPress
─────────────────────────────

Via WordPress Admin (Recommended):
  1. Log in to WordPress Admin Dashboard
  2. Go to: Plugins → Add New
  3. Click: "Upload Plugin"
  4. Select: loungenie-portal.zip
  5. Click: "Install Now"
  6. Wait for installation to complete
  7. Click: "Activate Plugin"


STEP 3: Configure the Plugin
──────────────────────────────

  1. After activation, go to WordPress Dashboard
  2. New menu item appears: "LounGenie Portal"
  3. Click: Settings / Configuration
  4. Configure:
     • Email settings (if using email features)
     • API credentials (if needed)
     • Portal settings
  5. Click: Save


STEP 4: Verify Installation
─────────────────────────────

  1. Go to: /portal/ (you'll see the portal interface)
  2. Check System Health in plugin settings
  3. All lights should be green ✅
  4. Portal is now active!
