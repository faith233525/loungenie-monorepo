To deploy this plugin:

1. Copy plugin directory to WordPress plugins folder
   Location: /wp-content/plugins/loungenie-portal/

2. Activate plugin in WordPress admin
   Dashboard → Plugins → Find "LounGenie Portal" → Activate

3. Configure plugin settings (optional)
   Dashboard → LounGenie Portal → Settings

4. Verify installation
   Check Dashboard → LounGenie Portal → System Health

5. Test functionality
   Visit /portal/ to see the portal interface
