HOW TO PREPARE FOR WORDPRESS UPLOAD
────────────────────────────────────

Option 1: ZIP the Plugin Directory
──────────────────────────────────

1. Keep this folder structure intact:
   loungenie-portal/
   ├── loungenie-portal.php
   ├── uninstall.php
   ├── readme.txt
   ├── includes/
   ├── api/
   ├── templates/
   ├── assets/
   ├── roles/
   └── wp-cli/

2. Create a ZIP file:
   - File name: loungenie-portal.zip
   - Should contain the "loungenie-portal" folder

3. Upload to WordPress:
   Dashboard → Plugins → Add New → Upload Plugin → Select ZIP

Option 2: Manual Upload via FTP
────────────────────────────────

1. Connect to your server via FTP
2. Navigate to: /wp-content/plugins/
3. Create folder: loungenie-portal
4. Upload all files and folders maintaining structure
5. Activate in WordPress Admin

═════════════════════════════════════════════════════════════════

WHAT NOT TO UPLOAD
───────────────────

DO NOT include these in your upload:

❌ Documentation files
   - *.md files (Markdown)
   - *.txt files (except readme.txt)
   - DEPLOYMENT-*.md
   - *-GUIDE.md
   - etc.

❌ Test files
   - /tests/ directory
   - *-TEST*.php
   - *-tests.php
   - phpunit.xml

❌ Development files
   - .git/ directory
   - .gitignore
   - .editorconfig
   - .eslintrc.json
   - composer.json
   - composer.lock
   - /bin/ directory
   - /vendor/ directory (if Composer not used)

❌ Demo/Preview files
   - PORTAL_PREVIEW.html
   - preview-*.php
   - demo-*.php

❌ Configuration files
   - phpcs.xml
   - package.json
   - Various config files

ONLY THE ITEMS LISTED IN THE DIRECTORY STRUCTURE ABOVE!

WORDPRESS INSTALLATION
──────────────────────

7. Start using the portal
