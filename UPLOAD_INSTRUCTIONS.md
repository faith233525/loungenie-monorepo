## ✅ TROUBLESHOOTING

### "Plugin not showing in list"
→ Go back to: Plugins → Add New → Upload Plugin  
→ Try uploading ZIP again

### "Portal page is blank"
→ Make sure **Pretty Permalinks** are enabled:
   - Settings → Permalinks
   - Select any option (Post name recommended)
   - Click Save Changes

### "Getting 404 error"
→ Flush permalinks by going to:
   - Settings → Permalinks
   - Click "Save Changes" without changing anything
