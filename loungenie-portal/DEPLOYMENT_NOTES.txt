INSTALLATION STEPS
------------------
1. Backup WordPress (database + files)
2. Upload ZIP to /wp-content/plugins/
3. Extract: loungenie-portal-production/
4. Rename to: loungenie-portal/
5. Activate plugin in WordPress Admin
6. Go to Settings → Permalinks → Save Changes (flush rules)
7. Test: Visit yoursite.com/portal
