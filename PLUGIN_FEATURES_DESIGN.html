What is LounGenie Portal?

LounGenie Portal is a commercial WordPress plugin that provides a complete enterprise SaaS partner management system. It features:

  • Theme-Independent Design - Works with any WordPress theme
  • Secure Authentication - OAuth 2.0 and WordPress user management
  • Role-Based Access Control - Support and Partner roles
  • REST API - Complete REST endpoints for all operations
  • Email Integration - Microsoft Graph and Outlook integration
  • CRM Sync - HubSpot integration with auto-sync
  • Database Schema - Companies, units, tickets, service requests

REST API Endpoints

All endpoints require authentication and return JSON responses. Base URL: /wp-json/lgp/v1/

Tickets Endpoint

GET /tickets
List tickets (filtered by role)
GET /tickets/{id}
Get single ticket details
POST /tickets
Create new ticket (Partners)
PUT /tickets/{id}
Update ticket (Support only)
POST /tickets/{id}/reply
Add reply to ticket thread

Companies Endpoint

GET /companies
List all companies (Support only)
GET /companies/{id}
Get company details
POST /companies
Create company (Support only)
PUT /companies/{id}
Update company (Support only)

Units Endpoint

GET /units
List units (filtered by role)
GET /units/{id}
Get unit details
POST /units

Update unit (Support only)

Dashboard Endpoint

GET /dashboard
Get dashboard metrics (aggregated by role)

Attachments Endpoint

Database Schema

Core Tables (4)

lgp_companies

Column Type Description
id INT PRIMARY KEY Company ID
name VARCHAR(255) Company name
address TEXT Company address
phone VARCHAR(20) Contact phone
email VARCHAR(255) Contact email
management_company_id INT FK Parent management company
created_at DATETIME Creation timestamp
updated_at DATETIME Last update timestamp

lgp_units

Column Type Description
id INT PRIMARY KEY Unit ID
company_id INT FK Associated company
address TEXT Unit location
lock_type VARCHAR(50) Lock type (combination, key, card)
color VARCHAR(20) Unit color tag
status VARCHAR(50) Status (active, install, service)
install_date DATE Installation date
created_at DATETIME Creation timestamp

lgp_tickets

Column Type Description
id INT PRIMARY KEY Ticket ID
service_request_id INT FK Associated service request
title VARCHAR(255) Ticket subject
description LONGTEXT Ticket content
status VARCHAR(50) Status (open, in-progress, closed)
priority VARCHAR(20) Priority (low, medium, high)
assigned_to INT FK Assigned user
internet_message_id VARCHAR(255) Email deduplication
created_at DATETIME Creation timestamp

lgp_service_requests

Column Type Description
id INT PRIMARY KEY Service request ID
company_id INT FK Associated company
type VARCHAR(50) Type (install, update, maintenance, repair)
status VARCHAR(50) Status (pending, in-progress, completed)
priority VARCHAR(20) Priority level
notes LONGTEXT Internal notes
created_at DATETIME Creation timestamp
