METHOD 1: ZIP Upload (Easiest)
──────────────────────

1. Zip the plugin folder:
   Folder Name: loungenie-portal
   Contents: All 9 items above

2. Filename: loungenie-portal.zip

3. Upload via WordPress:
   Admin → Plugins → Add New → Upload Plugin
   Select loungenie-portal.zip
   Click "Install Now"

4. Activate:
   Plugins → Find "LounGenie Portal" → Activate

METHOD 2: FTP Upload
────────────────────

1. Connect to server via FTP

2. Navigate to: /wp-content/plugins/

3. Create folder: loungenie-portal

4. Upload all 9 items into this folder

5. Activate in WordPress Admin
